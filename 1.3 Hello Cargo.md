# Compile then run
Compile: `rustc main.rs`

Run: `.\main.exe`

# Compile and run with cargo
`cargo run main`

# Compile with cargo - dev
Compile: `cargo build`

Run: `.\target\debug\thebook.exe`

# Check whether program will compile
`cargo check`

# Build for release
Compile: `cargo build --release`

Run: `.\target\release\thebook.exe`
